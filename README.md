# naming README

这是一个帮助中国程序员规范Class，Function……命名的工具。比如输入"工单系统"，再输入一个"/"号，会触发自动命名的翻译函数，出现三种不同的函数命名提示。

1. 输入想要翻译的中文,比如"工单系统"；
2. 输入"1"，比如"工单系统1"
3. 会出现翻译后的函数列表
4. 选择其中一个函数，会自动替换中文。


> Tip: 在联网状态下才能使用。
